# Project Jasa Wrapping Kendaraan

## Step by Step:

1. Clone repo ini.
2. Jalankan `composer install`.
3. Jalankan `npm install && npm run build`.
4. Copy `.env.example` menjadi `.env`.
5. Sesuaikan `DB_PORT`, `DB_DATABASE`, dll di `.env`.
6. Jalankan `php artisan key:generate`.
7. Jalankan `php artisan migrate`.
